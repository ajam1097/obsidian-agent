# Starter Skills

This starter keeps skills in one folder for easy copying:

```text
skills/
  idea-research/
  project-audit/
  session-close/
  process-new-sources/
```

These are intentionally smaller than private production skills. They show the operating pattern:

- read the vault,
- respect schemas,
- propose judgment changes before writing,
- keep durable state current.

## Included Skills

- `idea-research` — shape an idea with focused research without promoting it automatically.
- `project-audit` — review project state and drift without automatic repairs.
- `process-new-sources` — turn captured source pages into concept-page suggestions.
- `session-close` — preserve durable session state without routine logs.
