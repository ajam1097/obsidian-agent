# Sample Vault Agent Instructions

This sample vault demonstrates schema-governed agent workflows.

## Rules

- Read freely.
- Preserve source bodies.
- Propose project status changes before writing them.
- Write only valid frontmatter in governed folders.
- Keep personal examples fake and generic.

## Folders

- `efforts/ideas/` — uncommitted possibilities.
- `efforts/projects/` — committed outcomes.
- `sources/` — captured source material.
- `wiki/topics/` — synthesized concepts.
- `systems/` — current-state setup notes.
