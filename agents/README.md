# Starter Agents

Agents are optional. Many users can get the value of this starter with skills alone.

If you add agents, keep them narrow:

- `librarian` — reads and audits vault structure, proposes repairs.
- `researcher` — processes sources into concept-page suggestions.
- `project-steward` — reviews project state and prepares closeout updates.

Do not give an agent broad write authority until the matching schema audits are clean and the pre-write gate is proven in a sample vault.
